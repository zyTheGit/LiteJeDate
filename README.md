# LiteJeDate
只需要引用一个文件的日期插件

## [jeDate V6.5.0 日期控件](http://www.jemui.com/)

## 页面使用
```
# html
<input type='text' class='test'>

#js
<script src='./dist/LiteJeDate.js'></script>	
 jeDate(".test",{
    format:"YYYY年MM月DD日 hh:mm",
    isTime:true,
    festival: true,//开启万年历
    minDate:"2014-09-19 00:00:00"
})
```
